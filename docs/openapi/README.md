# Документация OpenAPI

Эта директория содержит спецификации OpenAPI для различных API в рекламной платформе.

## Доступные спецификации

- [Ads API](./ads-api-openapi.yaml) - Основное API для управления рекламными кампаниями и объявлениями
- [Auth API](./auth-api-openapi.yaml) - API аутентификации и биллинга

## Что такое OpenAPI файлы?

Файлы с расширением `.yaml` в этой директории содержат спецификации API в формате OpenAPI. Это стандартный язык описания RESTful API, который позволяет:

- Понимать возможности API без изучения исходного кода
- Генерировать клиентские библиотеки для разных языков программирования
- Создавать серверные заглушки
- Визуализировать API с помощью таких инструментов, как Swagger UI
- Автоматически тестировать API

### Файл ads-api-openapi.yaml
Этот файл описывает API сервиса объявлений (написанного на Go), включая все его конечные точки:
- Методы для создания, получения, обновления и удаления рекламных кампаний
- Методы для работы с отдельными объявлениями
- Эндпоинты проверки состояния и метрик

### Файл auth-api-openapi.yaml
Этот файл описывает API сервиса аутентификации и биллинга (написанного на Python), включая:
- Методы для пополнения и списания средств
- Получение баланса пользователя
- История транзакций
- Операции по начислению за создание объявлений и креативов

Таким образом, эти файлы служат технической документацией для разработчиков, которые хотят интегрироваться с нашими API, а также позволяют автоматически генерировать код для взаимодействия с API.
